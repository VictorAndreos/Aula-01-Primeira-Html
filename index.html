Sinal didático, com parêntese (/Exemplo) é pra poder exemplificar o encerameno de uma tag.

  1. (!DOCTYPE html) Inicio do código
  2. (HTML lang="en")
  3. (HEAD) Cabeça do código
  4. ....(Meta charset="UTF-8")
  5. ....(Meta http-equiv="X-UA-Compatible" content="IE=edge")
  6. ....(Meta name="viewport" content="width=device-width, initial-scale=1.0")>
  7. ....(Title) Nome do seu projeto (/Title)
  8. (/HEAD)
  9. (BODY)Corpo do código
  10. ....(H1) Sub titulo(/H1)
  11. (/BODY)
  12. (/HTML)
